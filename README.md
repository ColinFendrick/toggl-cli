# Usage
<!-- usage -->
```sh-session
$ npm install -g toggl-cli
$ toggl-cli COMMAND
running command...
$ toggl-cli (-v|--version|version)
toggl-cli/0.0.0 darwin-x64 node-v11.10.1
$ toggl-cli --help [COMMAND]
USAGE
  $ toggl-cli COMMAND
...
```
